# 📚 Library API

## Objetivo

API REST simples para gerenciamento de livros, desenvolvida para praticar conceitos de backend com Spring Boot.

## Tecnologias

- Java
- Spring Boot
- Spring Data JPA
- Maven
- Banco de dados (H2 ou MySQL)

## Endpoints

Criar livro

POST /books

Listar livros

GET /books

Buscar livro por ID

GET /books/{id}

Deletar livro

DELETE /books/{id}
